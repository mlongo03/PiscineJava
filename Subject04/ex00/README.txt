PROJECT DESCRIPTION:

this project has to convert a bmp two-colored image (only black and white) in a rappresentation of its own on the standard output where an ASCII character will rappresent
black pixels and another one white pixels, the character for white pixel and the character for the black pixel are get from the input parameters and the path of the file
aswell in particular the form of the the excecution will be explained.

COMPILATION:

javac --release 8 -d ./target ./src/java/edu.Roma42.printer/*/*.java

EXCECUTION:

java -cp ./target edu.Roma42.printer.logic.Program <black_char> <white_char> <path_bmp>
